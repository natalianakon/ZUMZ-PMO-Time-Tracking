PMO Time Tracker
Sharing with the Team

Read-only access · No install needed
What
A live PMO dashboard — shareable via Claude
The PMO Time Tracker is a React app that lives on Natalia's machine. It tracks hours by person, project, and work type. The full app ships as a single .jsx file — paste it into Claude, it renders instantly.
Why
Visibility without needing server access
Cara and Quin can view real team data — Dashboard, Exec View, Team breakdown, Projects — without needing to run the app locally. Natalia exports a JSON snapshot weekly to keep the data fresh.
How
Two files + one Claude prompt
Natalia sends the .jsx file + a JSON backup. You paste the JSX into Claude.ai, ask it to render as an artifact, then hit ↑ Restore to load the data. Full dashboard in under a minute.
Step-by-step
1
Natalia · Weekly
Export JSON backup
Click ↓ Backup in the tracker. Share the .json + latest .jsx file via Teams.
2
Cara & Quin
Open Claude.ai
Go to claude.ai or Claude in Teams. Start a new chat.
3
Cara & Quin
Upload the JSX & ask Claude
Attach the .jsx file and type: "Render this React component as an artifact."
4
Cara & Quin
Load the JSON data
Once rendered, click ↑ Restore in the app and upload the .json file Natalia sent.
5
Cara & Quin
Explore freely 🎉
Browse Dashboard, Exec View, Team, Projects. Read-only — nothing you do affects Natalia's live data.
